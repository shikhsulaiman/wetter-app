# 🌤️ Wetter-App

Moderne Wetter-App mit Echtzeit-Daten für jede Stadt der Welt.

## 🚀 Live-Demo
👉 https://shikhsulaiman.github.io/wetter-app/

## ✨ Features
- Stadt-Suche weltweit
- Aktuelle Temperatur, Luftfeuchtigkeit, Wind
- Wetter-Beschreibung mit Emojis
- Fehlerbehandlung (Stadt nicht gefunden, Netzwerk-Fehler)
- Enter-Taste & Button
- Modernes Glasmorphism-Design

## 🛠️ Technologien
- HTML5, CSS3 (Flexbox, Gradient, Backdrop-Filter)
- JavaScript (Vanilla, kein Framework)
- Open-Meteo API (Geocoding + Wetter)
- Fetch API mit Promise-Chaining

## 📚 Was ich gelernt habe
- Zwei APIs verketten (Geocoding → Wetter)
- DOM-Manipulation mit JavaScript
- Asynchrone Programmierung mit fetch().then()
- Fehlerbehandlung mit .catch()
- Responsives Design
